Dekompozycja układu mostkowego tylko przez R

Instrukcja liczenia niezawodności układu z pięcioma elementami wyłącznie przez R. Wszystkie uproszczenia, zwarcia, rozwarcia i wzory zapisujemy przez niezawodności R oraz wyrażenia typu 1 - R.

Rozwarcie elementu Ri: prawdopodobieństwo = 1 - Ri
Rukładu = P(sygnał przejdzie z wejścia do wyjścia)

1. Zasady upraszczania

Szeregowo liczymy przez R
R = Ra · Rb
Układ działa tylko wtedy, gdy działają oba elementy.
Równolegle też zapisujemy przez R
R = 1 - (1 - Ra)(1 - Rb)
Układ nie działa tylko wtedy, gdy nie działa żadna gałąź.
Dekompozycja po elemencie k
R = RkR(1k) + (1 - Rk)R(0k)
Czyli: element działa albo element jest uszkodzony.

2. Najwygodniejsza dekompozycja po elemencie R5

Przypadek 1: R5 działa

Węzły środkowe łączą się w jeden punkt.

Lewo: R1 || R4

Prawo: R2 || R3

R(15) = [1 - (1-R1)(1-R4)] · [1 - (1-R2)(1-R3)]

Przypadek 2: R5 nie działa

Zostają dwie gałęzie równoległe.

Góra: R1 · R2

Dół: R4 · R3

R(05) = 1 - (1 - R1R2)(1 - R4R3)

R = R5[1 - (1-R1)(1-R4)][1 - (1-R2)(1-R3)]
+ (1-R5)[1 - (1-R1R2)(1-R4R3)]

3. Ta sama metoda, ale wszystko zapisane przez R

Równoległe zapisane przez R

R14 = 1 - (1-R1)(1-R4)
R23 = 1 - (1-R2)(1-R3)
R(15) = R14 · R23

Metoda przez R dla szeregowych

R12 = R1R2
R43 = R4R3
R(05) = 1 - (1-R12)(1-R43)

Gdy gałęzie są równoległe, zapisujemy brak przejścia gałęzi jako nawias typu 1 - R, ale cały wzór dalej jest zapisany wyłącznie przez R.

4. Dekompozycja po każdym elemencie

Każdy element może być wybrany jako element dekompozycji. Poniżej są gotowe wzory dla elementów 1-5 oraz rysunek układu po zwarciu i rozwarciu danego elementu. Wszystko zapisane jest przez R oraz wyrażenia 1 - R.

R1
Zwarcie R1: element działa
Rozwarcie R1: element nie działa
Krok 1: wzór dekompozycji po R1
R = R1R(11) + (1-R1)R(01)
Krok 2: R1 zwarte, więc liczymy R(11)
R45 = 1 - (1-R4)(1-R5)
R345 = R3R45
R(11) = 1 - (1-R2)(1-R345)
Krok 3: podstawienie R(11)
R(11) = 1 - (1-R2)(1-R3[1-(1-R4)(1-R5)])
Krok 4: R1 rozwarte, więc liczymy R(01)
R52 = R5R2
R3||52 = 1 - (1-R3)(1-R52)
R(01) = R4R3||52
Krok 5: pełny wzór po R1
R = R1[1 - (1-R2)(1-R3[1-(1-R4)(1-R5)])]
+ (1-R1)[R4(1-(1-R3)(1-R5R2))]
R2
Zwarcie R2: element działa
Rozwarcie R2: element nie działa
Krok 1: wzór dekompozycji po R2
R = R2R(12) + (1-R2)R(02)
Krok 2: R2 zwarte, więc liczymy R(12)
R35 = 1 - (1-R3)(1-R5)
R435 = R4R35
R(12) = 1 - (1-R1)(1-R435)
Krok 3: podstawienie R(12)
R(12) = 1 - (1-R1)(1-R4[1-(1-R3)(1-R5)])
Krok 4: R2 rozwarte, więc liczymy R(02)
R15 = R1R5
R4||15 = 1 - (1-R4)(1-R15)
R(02) = R3R4||15
Krok 5: pełny wzór po R2
R = R2[1 - (1-R1)(1-R4[1-(1-R3)(1-R5)])]
+ (1-R2)[R3(1-(1-R4)(1-R1R5))]
R3
Zwarcie R3: element działa
Rozwarcie R3: element nie działa
Krok 1: wzór dekompozycji po R3
R = R3R(13) + (1-R3)R(03)
Krok 2: R3 zwarte, więc liczymy R(13)
R25 = 1 - (1-R2)(1-R5)
R125 = R1R25
R(13) = 1 - (1-R4)(1-R125)
Krok 3: podstawienie R(13)
R(13) = 1 - (1-R4)(1-R1[1-(1-R2)(1-R5)])
Krok 4: R3 rozwarte, więc liczymy R(03)
R45 = R4R5
R1||45 = 1 - (1-R1)(1-R45)
R(03) = R2R1||45
Krok 5: pełny wzór po R3
R = R3[1 - (1-R4)(1-R1[1-(1-R2)(1-R5)])]
+ (1-R3)[R2(1-(1-R1)(1-R4R5))]
R4
Zwarcie R4: element działa
Rozwarcie R4: element nie działa
Krok 1: wzór dekompozycji po R4
R = R4R(14) + (1-R4)R(04)
Krok 2: R4 zwarte, więc liczymy R(14)
R15 = 1 - (1-R1)(1-R5)
R215 = R2R15
R(14) = 1 - (1-R3)(1-R215)
Krok 3: podstawienie R(14)
R(14) = 1 - (1-R3)(1-R2[1-(1-R1)(1-R5)])
Krok 4: R4 rozwarte, więc liczymy R(04)
R53 = R5R3
R2||53 = 1 - (1-R2)(1-R53)
R(04) = R1R2||53
Krok 5: pełny wzór po R4
R = R4[1 - (1-R3)(1-R2[1-(1-R1)(1-R5)])]
+ (1-R4)[R1(1-(1-R2)(1-R5R3))]
R5
Zwarcie R5: element działa
Rozwarcie R5: element nie działa
Krok 1: wzór dekompozycji po R5
R = R5R(15) + (1-R5)R(05)
Krok 2: R5 zwarte, więc środkowe węzły są jednym punktem
R14 = 1 - (1-R1)(1-R4)
R23 = 1 - (1-R2)(1-R3)
R(15) = R14R23
Krok 3: podstawienie R(15)
R(15) = [1-(1-R1)(1-R4)][1-(1-R2)(1-R3)]
Krok 4: R5 rozwarte, więc zostają dwie gałęzie
R12 = R1R2
R43 = R4R3
R(05) = 1 - (1-R12)(1-R43)
Krok 5: pełny wzór po R5
R = R5[1-(1-R1)(1-R4)][1-(1-R2)(1-R3)]
+ (1-R5)[1-(1-R1R2)(1-R4R3)]

5. Ostateczny wzór dla różnych R1...R5

Najładniejsza postać, przez dekompozycję po R5

R = R5(R1 + R4 - R1R4)(R2 + R3 - R2R3)
+ (1-R5)(R1R2 + R4R3 - R1R2R4R3)

Postać rozwinięta

R = R1R2 + R3R4 + R1R3R5 + R2R4R5
- R1R2R3R4 - R1R2R3R5 - R1R2R4R5
- R1R3R4R5 - R2R3R4R5 + 2R1R2R3R4R5

Gdy wszystkie elementy mają tę samą niezawodność

R1 = R2 = R3 = R4 = R5 = R
RUKŁADU = 2R2 + 2R3 - 5R4 + 2R5

Jeżeli elementy mają różne niezawodności, zostawiasz wzór ogólny i podstawiasz osobne wartości R1, R2, R3, R4, R5. Jeżeli wszystkie są takie same, używasz krótszego wzoru powyżej.
